# SkaiConnect Static Website Redesign

This folder contains a complete static HTML redesign for SkaiConnect using a premium black, red, white, and light-grey visual direction.

## Preview

The local preview server is running at:

http://127.0.0.1:8080/

If the server is not running, start it from this folder with any static server. For example:

```powershell
C:\Users\TECHNIFI\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe -m http.server 8080 --bind 127.0.0.1
```
